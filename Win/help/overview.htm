Windows Frotz is an interpreter for all of Infocom's games and all other games compiled for the Z-Machine. Great care has been taken to ensure that Windows Frotz complies fully with and implements all features described in The Z-Machine Standard, Version 1.1, by Graham Nelson, et al.

The features of Windows Frotz include:

  • Full support for all Z-code versions V1-V8, including V6 ("graphical").
  • Fully compliant with The Z-Machine Standard, Version 1.1.
  • Complete Blorb version 2.0 support.
  • Colour and graphics (Blorb PNG and JPEG, plus Kevin Bracey's extensions for Infocom's V6 games).
  • Alpha blending and gamma correction for PNG graphics.
  • Sound and music (Blorb AIFF, MOD and Ogg Vorbis).
  • Full support for foreign languages in both input and output, including full Unicode support.
  • Timed input, mouse input and menu support.
  • Optional strict error checking.
  • Machine and interpreter independent Quetzal format save files.
The latest release of Windows Frotz is available from the Frotz page at the IF Archive. The IF Archive also contains hundreds of games that can be played with Windows Frotz.
